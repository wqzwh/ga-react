# react 单页面工程脚手架

集成 react+redux+redux-saga+react-router+antd+lodash+moment+axios+less

> react project

## 基本命令

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
npm run start

# build for production with minification
npm run build

# lint检测命令
npm run lint

# 项目包大小分析命令
npm run analyz
```
